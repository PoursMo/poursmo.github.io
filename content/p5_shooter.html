After making some very small projects on p5.js Web Editor to learn JavaScript, I started making a game using this same library.
The game is a shoot 'em up with difficulty increasing over time. At 5 minutes, the boss spawns and defeating it rewards the player with victory.
The p5 library has very basic features, so I had to code most functions myself - like collision detection or sprite splicing - which was challenging and fun.
